# Dashboard Register Shortcut

Adds the Register as a first-class daily operations area instead of leaving it buried under Staff.

## Included

- New main sidebar item: Register.
- New standalone Register screen using the existing safe Register/POS component.
- Dashboard Today's Register card with:
  - net incoming
  - expected cash drawer
  - Cash / Clover / Venmo / PayPal / Check totals
  - quick actions for New Sale, Sell Credits, Record Payment, Cash Payout, and Close Day
- Dashboard Widget Controls now include Today's Register.

## Safety

No database changes, no data migrations, no destructive commands. Existing Staff → Register remains as a fallback.
